# 📖 Project Overview

The **Rule-Based AI Chatbot** is the first project of the **DecodeLabs Artificial Intelligence Industrial Training Program 2026**.

This chatbot demonstrates the fundamentals of Artificial Intelligence by using **rule-based logic**, **decision making**, and **control flow** instead of Machine Learning.

The chatbot interacts with users through predefined rules, processes user queries, performs calculations, maintains conversation history, and provides information related to Artificial Intelligence, Python, Machine Learning, and programming concepts.

